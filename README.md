# Sana ❤️🥳

A cinematic, emotional love letter for Sana with elegant animations and music.

## ✨ Features

- **Elegant Design**: Black & white theme with gold and rose accents
- **Perfect Animations**: Smooth fade-ins, slide transitions, and hover effects using cubic-bezier timing
- **Music Integration**: 
  - Lady Gaga - "Die With a Smile" (main cinematic soundtrack)
  - Alex Warren - "Ordinary" (background music at 20-30% volume)
- **Multiple Sections**: Connection, Moments, Love CV, Letter, Confession, and Final Question
- **Interactive Modal**: Yes/No response dialog
- **Celebration Effect**: Confetti animation on "Yes"
- **Responsive Design**: Perfect on mobile and desktop
- **Parallax Scroll**: Background moves with scroll for cinematic effect

## 📖 Sections

1. **Hero** - Beautiful intro with call-to-action
2. **Connection** - First impressions and calmness
3. **Moments** - Details noticed and appreciated
4. **Love CV** - Why serious and commitment
5. **Letter** - Deep feelings and emotions
6. **Confession** - Love declaration
7. **Final Question** - The proposal with modal response

## 🚀 How to Use

1. Open `index.html` in a web browser
2. Music plays automatically after 1.5 seconds
3. Scroll through the sections smoothly
4. Click "Your answer matters" to respond to the final question
5. Choose "Yes, I Will" or "Maybe Not"
6. Enjoy the celebration confetti if she says yes!

## 🎵 Music Setup

The website uses audio elements for background music. To add your actual music:

1. Replace the audio URLs in the HTML:
   - Main audio: Lady Gaga - "Die With a Smile"
   - Background audio: Alex Warren - "Ordinary"
2. Or use local audio files by changing the `<source src="...">` paths

## 🎨 Customization

- **Colors**: Edit CSS variables in `:root` for custom color schemes
- **Text**: Modify any section content directly
- **Font**: Change Playfair Display and Lora imports in the CSS
- **Animation Timing**: Adjust cubic-bezier values for different easing
